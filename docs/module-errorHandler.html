# inner handleErrors(err, history)

A standardized error handler to handle errors from requests to backend. If error.response.status is 401 and err.response.data is "authorized user", it means that the user is not signed in. User details will be removed from localStorage and a LOG_OUT_USER action will be dispatched. The user will then be redirected to the home page to login again.
